How to Use

Welcome to Plaid. Here's how to use it.

To split a row, hover over a region and press 'R'.

To split a column, hover over a region and press 'C'.

Right-click and select Row to split many at once.
Scroll or use the arrow keys to select the number of rows. Click to insert. (Escape key to cancel.)

Right-click and select Column to split many at once.
Scroll or use the arrow keys to select the number of columns. Click to insert. (Escape key to cancel.)

To merge, click on a region to select it. Move the mouse to another region and click.

To extend, click on the area near the bottom of the grid.

To mark content regions, use the Plaid Squares Palette.
Drag a content item to a region to mark it.

Click on the color square to change the HTML5 content type.

Click twice on the text to rename the content item's class.

To zoom in and out for a better view of your work, enter full screen mode.

Zoom by pinching in and out on the trackpad, or use the View application menu.

Click on the dots at the top of the screen to change layouts.

Adjust the breakpoints by dragging the breakpoint sliders.
Resize the window to see the layout change.

Adjusting breakpoints must be done in window mode (not full screen).

To export, use the File application menu and choose Export. There must be assigned content in order to export.

Choose a directory to export to. The HTML/CSS/JS will be placed in this directory.

Note: Only content regions will be exported; the grid itself is only an abstraction in Plaid and will not show up in the resulting HTML/CSS.

Double-click the exported index.html file in the Finder to view the layout in the browser.
